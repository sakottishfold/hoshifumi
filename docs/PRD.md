# PRD: ほしふみ

> Product Requirements Document
> Last updated: 2026-05-16
> 改名履歴:みっつ → いとなみ(2026-05-14, ADR-011/015) → ほしふみ(2026-05-16, ADR-018、worldview 言語化後)

## 1. Executive Summary

**ほしふみ (hoshifumi)** は寝る前のジャーナリングアプリ ─ **自分と向き合うための5分の儀式**である。各セッションは固定の scaffolding(身体感覚タップ、今日の出来事、明日への closure)と、ユーザーが書いた内容に基づいて「なぜそう感じたか」を尋ねる AI 生成の follow-up を組み合わせる。蓄積された記録に対して、AI はユーザー自身では気づきにくいパターンを surface する。

**Tagline**:寝る前の5分、星をふみに。

ターゲットは、ジャーナリングしたいが従来のジャーナリングのハードルが高くて続かない層。**寝る前という習慣にアンカーし(既存の睡眠ルーチンに habit stacking)**、具体的な scaffolding(白紙ではない)+ AI 主導の単一内省プロンプトを提供することで、「何を書けばいいか」のparalysisと、「一言だけ書いて終わった」trap の両方を解消する。

## 2. Problem Statement

### ユーザー
日本の25-45歳の成人で:
- 自分をもっと理解したい
- ジャーナリングアプリ(Apple Journal、Day One、Notion テンプレ、紙)を試したことがある
- 習慣化に失敗している(典型パターン:3日でやめる)
- スマホを毎日使う
- 自己改善ツールに月¥500-1000払う余裕がある

### Pain
1. **白紙の不安**:「何を書けばいい?」で始まる前に止まる
2. **時間的プレッシャー**:ジャーナリングには15分以上必要だと思っている
3. **フィードバックループなし**:書いても見返さない
4. **アプリが rigid すぎる or flexible すぎる**:Daylio はテンプレ固定、Notion はデザイン作業を強制

### なぜ今なら解けるか
- AI(Claude)が白紙のフォームではなくパーソナライズされた問いを返せる
- AI が短い構造化された入力からパターンを surface できる
- LLM コストが下がって consumer SaaS の経済が成立する

### なぜ ChatGPT/Claude を直接使うのではダメか
- 永続的な記録もレビュー UI もない
- 日次 nudge メカニズムなし
- パターン分析のための構造化データなし
- 30秒の日次入力のための専用 UI なし

## 3. Goals & Non-Goals

### v1.0 Goals
- セッション中央値 約5分(寝る前習慣にアンカー)、ただし下限はゼロ ─ スキップ日は明示的に OK
- 7日 retention >= 60%(クローズドβ)
- 30日 retention >= 30%(クローズドβ)
- AI follow-up question が個別化されていて、ユーザーが promptなしには書かなかったことを surface できている
- 月次 AI レポートが有用な insight を出す

### v1.0 Non-Goals
- 重いメンタルヘルスサポート(セラピーではない、自殺念慮の対応は専門機関に委ねる)
- ソーシャル機能・シェア
- ゲーミフィケーション(streak カウンタはあるがバッジ・リーダーボードなし)
- 多言語(日本語のみ)
- ネイティブ iOS/Android アプリ(PWA のみ)

### Anti-Goals(絶対やらない)
- スキップ日でユーザーを罰する機能は **追加しない**
- 中毒性のある UX パターン(variable reward、infinite scroll)は **使わない**
- ユーザーデータの売却・共有は **しない**
- 診断を行う AI 機能は **追加しない**

## 4. ターゲットユーザー(ペルソナ)

### Primary:30代の働く親
- 32歳、既婚、未就学児1-2人、共働き
- スマホネイティブ、仕事で Notion/Slack 利用
- 「もっと自分を見つめたい」と自覚している
- 寝る前に5-10分ある
- 月¥500なら生活改善ツールに払う

### Secondary:個人開発者・クリエイター
- 28-40歳、独身またはパートナーあり、子供なし
- エンジニア、デザイナー、クリエイター
- Obsidian を使う、BiP/IndieHackers を追っている
- 数ヶ月単位のプロジェクトを通じた内面状態を追跡したい

### Tertiary:20代独身
- 22-28歳、都市部一人暮らし
- 孤独だが high-functioning
- セラピー・瞑想アプリ・フィットネスにすでに課金している
- 「今日の気持ちを置いておく場所」が欲しい

## 5. 差別化

| 競合 | 勝てる理由 |
|---|---|
| Apple Journal | 寝る前儀式のフォーカス、AI follow-up、Android 対応 |
| Day One | 軽量、AI 組み込み、安い、時間軸あり |
| Daylio | リッチテキスト + AI 分析、絵文字だけじゃない |
| Reflectly | 日本語ネイティブ UX、儀式 first の framing |
| Awarefy | より軽い、セラピー的ポジショニングを避ける |
| Gravity | 寝る前特化、scaffolded prompts(自由 chat ではない)|
| Notion テンプレ | 日次 nudge + AI サマリーが標準で入っている |
| ChatGPT 直 | 永続 UI、構造化データ、setup 不要 |

ウェッジ:**「自分と向き合うための5分の寝る前儀式 + AI による個別化された深さ、日本語」** ─ 現在この交点を所有しているプレイヤーはいない。

## 6. マネタイズ

### プラン(v1.0)
- **Free**:1テンプレート(Basic)、30日履歴可視、AI レポートなし
- **Pro(¥480/月)**:全5テンプレート + custom、無制限履歴、月次 AI レポート、検索
- **Premium(¥980/月)**:Pro + AI 対話モード、AI 質問サジェスト、Obsidian エクスポート

### Unit Economics(目標)
- **Pro プランの粗利**:85%+(AI コスト 月約¥60/active user)
- **CAC 目標**:<¥1,500(オーガニック + コンテンツマーケ、当初は有料広告なし)
- **LTV 目標**:¥12,000+(¥480 で24+ヶ月 retention)

### 損益分岐
- 固定費:¥8,000/月(Vercel + Supabase + ドメイン)
- 課金ユーザー20名で損益分岐
- 課金ユーザー500名 = ¥240k/月 MRR

## 7. 成功指標

### North Star
**WAU(直近7日で4日以上完了した weekly active users)**

「習慣化したユーザー」を捉える指標。casual triers ではない。

### サポート指標
- D1、D7、D30 retention カーブ
- 日次完了率(DAU / WAU)
- 完了時間(目標:中央値 約5分 ─ 寝る前儀式であってスピードではない)
- AI follow-up エンゲージメント率(ユーザーが返したか、その返答が Q2 より有意に長かったか)
- 月次レポート閲覧率(生成したものを読んでもらえてるか)
- Free → Paid 転換率
- Churn 率(月次)

### Validation gates
- **Phase 0(セルフテスト)**:オーナーが30日連続で使う、スキップなし
- **Phase 1(クローズドβ、5ユーザー)**:D7 ≥ 60%
- **Phase 2(パブリックβ、100ユーザー)**:D30 ≥ 30%、paid 転換 ≥ 5%
- **Phase 3(ローンチ)**:6ヶ月で active 1,000ユーザー

## 8. 制約

### プライバシー
- 全データは Supabase で暗号化(サーバー側暗号化は自動、free_text へのフィールドレベル暗号化は v2.0 で検討)
- 自由記述は ユーザー明示同意なしには第三者 LLM に送らない
- 月次 AI レポートは Anthropic API のみで処理(ユーザー opt-in なしに他 LLM プロバイダ使用禁止)

### 法的
- 初回ローンチは日本のみ(GDPR の複雑さ回避)
- 価格表示は JPY のみ
- プライバシーポリシーは APPI 準拠
- 規約に「これは精神医療ではない」 disclaimer

### 安全性
- ユーザー入力に明示的な自傷・自殺念慮が含まれる場合、レスポンスに **必ず**危機リソース情報を含める(よりそいホットライン:0120-279-338)
- AI は医学的・精神医学的アドバイスをしない
- ユーザー入力コンテキストでの prompt-injection 脆弱な AI 対話なし(API 送信前にサニタイズ)
