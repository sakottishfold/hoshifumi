## ログの書き方

毎日使ったあとで、何か違和感があったら:

1. 日付をメモ
2. 何が起きたか書く
3. 重さを評価:⚠️(軽微)/ 🚨(中)/ 🔥(重 = スキップしようかと迷ったレベル)
4. **今は直さない**。記録だけ。パターンは数週間かけて見えてくる。

7日後・14日後・21日後・30日後に振り返って、テーマトップ3を抽出する。
